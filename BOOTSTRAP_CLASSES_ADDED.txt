============================================
RESPONSIVE BEHAVIOR:
============================================

MOBILE (< 576px):
- Search bar: Hidden (d-none)
- Button text: Hidden (d-none)
- Sidebars: Hidden (d-none)
- All cards: Full width (w-100)

TABLET (576px - 1199px):
- Search bar: Visible (d-md-flex)
- Button text: Visible (d-sm-inline)
- Sidebars: Still hidden
- Cards: Full width

DESKTOP (1200px+):
- Search bar: Visible
- All text: Visible
- Sidebars: Visible (d-xl-block)
- 3-column grid layout
