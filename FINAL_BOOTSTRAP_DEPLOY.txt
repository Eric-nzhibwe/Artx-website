============================================
KEY BOOTSTRAP CLASSES USED:
============================================

RESPONSIVE DISPLAY:
- d-none d-md-flex: Hide mobile, show tablet+
- d-none d-xl-block: Hide until desktop (1200px+)
- d-flex: Flexbox
- w-100: Full width

LAYOUT:
- container-fluid: Full width container
- flex-grow-1: Allow growing
- flex-shrink-0: Prevent shrinking
- flex-fill: Fill space

SPACING:
- p-0, p-3: Padding
- m-0, mb-3: Margin
- gap-2, gap-3: Flex gap

ALIGNMENT:
- align-items-center: Vertical center
- justify-content-center: Horizontal center
- justify-content-between: Space between

COMPONENTS:
- btn btn-light: Button styling
- form-control: Input styling
- rounded-pill: Pill shape
- shadow-sm: Shadow effect
- bg-white: White background

============================================
RESPONSIVE BREAKPOINTS:
============================================

Bootstrap 5 Breakpoints:
- xs: < 576px (Extra small - Mobile)
- sm: ≥ 576px (Small - Mobile landscape)
- md: ≥ 768px (Medium - Tablet)
- lg: ≥ 992px (Large - Desktop)
- xl: ≥ 1200px (Extra large - Wide desktop)
- xxl: ≥ 1400px (Extra extra large)

Our Implementation:
- Mobile: < 1200px (single column, hidden sidebars)
- Desktop: ≥ 1200px (3-column grid, visible sidebars)

IF SOMETHING DOESN'T WORK:
============================================

ISSUE: Changes not visible
SOLUTION: 
- Clear browser cache completely
- Try incognito mode
- Check Render deployment logs

ISSUE: Layout broken
SOLUTION:
- Check browser console for errors
- Verify all CDN links loaded
- Check internet connection

ISSUE: Still see old layout
SOLUTION:
- Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- Close browser completely and reopen
- Clear site data in browser settings
