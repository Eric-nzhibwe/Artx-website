## 🔧 Technical Architecture

### Frontend Stack:
- Pure HTML5, CSS3, JavaScript (ES6+)
- No frameworks or dependencies
- Responsive design
- Modern CSS (Grid, Flexbox, Gradients)
- Font Awesome icons

### Data Layer:
- Browser localStorage
- JSON data structures
- Base64 image encoding
- Client-side persistence
- ~5-10MB capacity

### Key Algorithms:
1. **Scoring**: Base + (Range × WordRatio × Creativity)
2. **Voting**: NetVotes = Upvotes - Downvotes
3. **Rewards**: Milestone-based (every 5 upvotes)
4. **Rankings**: Sort by net votes + time period

### Data Models:
- Users (with currency, votes, rewards)
- Challenges (with scoring criteria)
- Submissions (with vote tracking)
- Influencers (with stats)
- Events (with prizes)
- Special Challenges (with types)
